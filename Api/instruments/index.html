Instruments

You can find a list of request body parameters and possible outcomes here.

The SDK will infer the type of the instrument, if not provided.

Request a card instrument#

const instrument = await cko.instruments.create({    token: 'tok_bzi43qc6jeee5mmnfo4gnsnera', // token obtainer form Checkout.Frames of the mobile SDKs});

Get an instrument#

const instrument = await cko.instruments.get('src_udfsqsgpjykutgs26fiejgizau');

Update an instrument#

const instrumentResponse = await cko.instruments.update('src_udfsqsgpjykutgs26fiejgizau', {    expiry_year: 2030,});

Delete an instrument#

const instrumentResponse = await cko.instruments.delete('src_udfsqsgpjykutgs26fiejgizau');
